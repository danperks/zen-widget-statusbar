# Widget Statusbar

`Widget Statusbar` is a configurable bottom widget bar built for Zen Browser.

It adds a lightweight statusbar-style strip to the bottom of the browser UI with configurable widgets for:

- website domain
- clock
- date
- time on page
- page memory
- app memory
- battery
- pomodoro timer

## Requirements

- Zen Browser
- [Sine](https://github.com/CosmoCreeper/Sine)

## Install

1. Open Sine in Zen.
2. Add the repo `danperks/zen-widget-statusbar`.
3. Enable the mod.

## Note About JavaScript Mods

This mod uses a Sine JavaScript script in addition to CSS.

If you install it directly from a repo or as a local mod, Sine may require `Allow unsafe JS` to be enabled. Once installed from the Sine store, that extra toggle should not be necessary.
